// 01 — Live HTML Editor
HTML-Live v1.6

HTML-Live exists because I needed it to exist. Working through large HTML files by hand — trying to find specific sections, understand what the code was actually doing, and see the result instantly without switching between applications — was slowing everything down. So I built a tool that solved all of that in one place.

The core idea is simple: write or paste your HTML on one side, see a live preview on the other, with no delay and no button to press. Any change you make appears in the preview immediately.

  • Live preview — renders as you type, no button press
  • Line numbers with active line tracking
  • HTMLHint debug panel — flags errors and warnings in real time
  • Inspect mode — click any preview element to jump to its source
  • Version rollback — up to 20 snapshots

// 04 — Image Deduplication

Dupe Remove Preview

Find and remove duplicate images from your library using perceptual hashing — a visual fingerprinting technique that catches duplicates across different compressions, crops and file sizes that filename matching misses entirely. Groups similar images together so you can review and decide what to keep.

  • Perceptual hashing — compares how images look, not their bytes
  • Group review — similar images clustered, bulk keep or delete per group
  • Drag to compare — view any two images side by side before deciding
  • Adjustable sensitivity threshold
  • Export kept images with optional filename prefix
  • 100% local — nothing uploaded, no account, no tracking

Getting Sound in 30 Seconds
1 — Press PlayHit ▶ PLAY / STOP. The drum sequencer fires immediately.
2 — Play a noteClick any key on the keyboard to trigger the lead synth.
3 — Program bassClick purple cells in BASS SEQ to toggle steps, pick notes with dropdowns.
4 — ExportHit ⏺ REC, play your pattern, then ⏹ SAVE WAV to download audio. Use ↑ MIDI to export pattern as MIDI.
Drum Sequencer — 7 Rows
Kick (BD)808 sine sweep with transient click. Classic four-on-the-floor or half-time.
Snare (SD)Tone + noise blend. Sits on beats 2 and 4 in the default pattern.
Clap (CP)Layered noise bursts with short reverb tail — classic electronic clap. Pink step colour. CP knob in mixer.
Hat (CH)Square wave cluster. All steps lit = 8th-note grid.
TomShort pitched sine with pitch drop.
RimHigh-pitched short click for rimshots.
Synth RowTriggers lead synth using arp memory. Combine with ARP HOLD for melodic sequences.
CLR DRUMS button clears only the drum rows. CLR ALL clears drums + bass sequencer simultaneously.
Lead Synth Engines
SuperSaw7 detuned sawtooth oscillators. Wide, lush trance/house lead.
Moog BassTwo saws + sub square. Fat analogue bass/lead.
FM LeadFM carrier + modulator. Metallic, DX7 flavour.
Bass Sequencer
Step ButtonsPurple cells. Click to toggle on/off.
Note PickersDropdown under each step to choose pitch.
CLRClears all bass steps instantly.
RNDRandomises steps and notes using pentatonic set.

Audio Export (WAV)
⏺ RECStarts recording the live audio output. The REC badge flashes at the top. Start playback before or after pressing REC.
⏹ SAVE WAVStops recording and downloads a stereo 44.1kHz WAV file of everything that was captured.
TIP: Record a full loop (4–8 bars), then stop. The WAV encodes instantly in the browser — no upload needed.
MIDI Export (↑ MIDI)
What's exportedAll drum hits (on standard GM channels), bass sequencer notes, and the synth arp memory are written as a Type-1 MIDI file at the current BPM.
ChannelsCh10 = drums (GM: Kick=36, Snare=38, Clap=39, Hat=42, Tom=48, Rim=37). Ch2 = bass. Ch1 = synth/arp.
MIDI Import (↓ MIDI)
How it worksOpens a file picker. Select any standard MIDI file. The importer reads the first track, maps notes to drum rows and bass notes, and programs the sequencer.
Note mappingGM drum notes (Ch10) map to rows. Non-drum notes on any channel go to the bass sequencer. Only the first stepLen steps are loaded.
NOTE: MIDI import works best with simple loop files (1–2 bars). Complex arrangements may only partially map to the 8/16-step grid.
Clear Sequencers
✕ CLR ALLClears all drum rows AND all bass steps simultaneously. Useful for starting fresh.
✕ CLR DRUMSClears only the 7 drum rows, leaves bass pattern intact.
CLR (bass)Clears only bass steps, leaves drum pattern intact.

DPAUL MKV — OPERATING MANUAL
Getting Sound in 30 Seconds
1 — Press PlayHit ▶ PLAY / STOP. The drum sequencer fires immediately.
2 — Play a noteClick any key on the keyboard to trigger the lead synth.
3 — Program bassClick purple cells in BASS LINE 1, orange cells in BASS LINE 2. Both fire simultaneously each step.
4 — Save presetClick ↓ SAVE on any preset slot to capture the current state. Name it, export as JSON.
Drum Sequencer — 7 Rows
Kick (BD)808 sine sweep with transient click.
Snare (SD)Tone + noise blend. Beats 2 and 4.
Clap (CP)Layered noise bursts with short reverb tail.
Hat (CH)Square wave cluster. All steps = 8th-note grid.
TomShort pitched sine with pitch drop.
RimHigh-pitched short click for rimshots.
Synth RowTriggers lead synth using arp memory. Combine with ARP HOLD for melodic sequences.
Lead Synth Engines
SuperSaw7 detuned sawtooth oscillators. Wide, lush trance/house lead.
Moog BassTwo saws + sub square. Fat analogue bass/lead.
FM LeadFM carrier + modulator. Metallic, DX7 flavour.
Dual Bass Sequencer
Line 1 (Purple)First bass voice. Both lines share the same engine, cutoff, reso and decay settings.
Line 2 (Orange)Second bass voice. Runs in parallel — great for counterpoint, octave doublings, or polyrhythm.
VOL 1 / VOL 2Independent volume knobs for each bass line in the sequencer header.
RND 1 / RND 2Randomise each line independently using a pentatonic set.
CLR 1 / CLR 2Clear each bass line independently.
AI TargetIn the AI Bass Gen panel, use LINE 1 / LINE 2 buttons to select which lane Claude generates into.
Bass Engines (shared by both lines)
SUB BASS
Pure sine + octave harmonic.
ACID 303
Sawtooth with resonant filter sweep.
SLAP BASS
Percussive pitch attack + sustain.
RUBBER
Pitch drop from 3× to target note.
REESE
Three detuned sawtooths. D&B classic.
CLICK BASS
Noise transient + short sine.

Audio Export
⏺ RECStarts recording the live audio output.
⏹ SAVEStops recording and downloads the audio file.
MIDI Export / Import
↑ MIDIBoth bass lines are exported: Line 1 on Ch2, Line 2 on Ch3. Drums on Ch10, synth on Ch1.
↓ MIDIImport a MIDI file. First non-drum MIDI track maps to Bass Line 1.
Memory Presets 3 SLOTS
↓ SAVECaptures the entire machine state into that slot: BPM, both drum and bass patterns (both lines), engine, all knob values.
↑ LOADRestores the machine to the saved state. Playback continues at new BPM if playing.
NamingClick the name field (e.g. "EMPTY SLOT") and type a name up to 20 characters. Names are saved with the preset.
✕ ClearErases the preset data from that slot. The name field returns to empty.
↓ EXPORT JSONDownloads a .json file containing all three preset slots. Share or back up your patterns.
↑ IMPORT JSONLoad a previously exported preset JSON file. Merges into all three slots.
TIP: Presets are stored in browser localStorage — they persist between sessions automatically. Export JSON to back them up or share with others.

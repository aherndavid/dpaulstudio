// 01 — Live HTML Editor
HTML-Live v1.6

HTML-Live exists because I needed it to exist. Working through large HTML files by hand — trying to find specific sections, understand what the code was actually doing, and see the result instantly — was slowing everything down.

The device preview modes were a priority — being able to switch between a mobile, tablet, and desktop view without leaving the editor is something that should be standard.

Version rollback is the one that saves the most frustration. If an edit breaks something, you can step back through previous states without losing anything.

  • Live preview — renders as you type
  • Line tracking & HTMLHint debug panel
  • Inspect mode — click to jump to source
  • Version rollback — up to 20 snapshots
  • Export formatted or minified HTML
  • Single-file — no install required

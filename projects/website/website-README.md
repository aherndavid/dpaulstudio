# Website Projects

HTML/CSS/JavaScript projects built and deployed via GitHub Pages.

Each project has a desktop and mobile version where applicable.

---

## Projects

### Acid Box
A browser-based synthesiser and drum machine interface — inspired by classic hardware like the Roland TB-303 and TR-808.

### CRT
A retro CRT monitor visual effect built in HTML/CSS.

### Monthly Expenditure
A clean, simple personal finance tracker for logging and visualising monthly spending.

### Moving Car
A CSS and JavaScript motion animation.

### Synth
An interactive web-based synthesiser.

### TTS (Text to Speech)
A text-to-speech interface with desktop and mobile versions.

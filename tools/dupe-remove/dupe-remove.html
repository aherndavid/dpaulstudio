How to Use // Dupe Remove
1
Import your images
Drag images into the drop zone, or use the FILES / FOLDER buttons to select from your computer.
2
Set sensitivity
Slide left for stricter matching (exact dupes only). Slide right to catch visually similar images too.
3
Run Analysis
Perceptual hashing compares every image. Results appear as groups — originals are marked green, duplicates red.
4
Review each group
Use KEEP ALL or DELETE DUPES per group, or click individual images to toggle their status. ✓ = keep · ✕ = delete.
5
Export or Purge
Add an optional prefix then hit EXPORT KEPT to download clean images. PURGE SESSION removes marked images from the current view.
⚠ No files are ever deleted from your disk — purge only removes from this session.
⚠ Exports go to your Downloads folder one file at a time to avoid browser limits.
⚠ TIFF files are not supported — browsers cannot decode them. Convert to JPG or PNG first.
⚠ End Session?

This will clear all loaded images, groups, and selections.

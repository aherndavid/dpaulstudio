How to Use // Dupe Remove
1
Import
Use FILES to pick individual images or FOLDER to load an entire directory.
2
Set sensitivity
Lower = stricter (exact dupes only). Higher = catches visually similar images too.
3
Run Analysis
Perceptual hashing groups similar images together. Review each group and mark what to keep or delete.
4
Review groups
Use KEEP ALL or DELETE DUPES per group, or click individual images to toggle. Green border = keep · Red border = delete.
5
Orient (optional)
Crop all images to a target ratio (16:9, 9:16, 1:1 etc.) and rotate individual images before exporting.
6
Manual Remove (optional)
Select individual images to remove from the session without running analysis — useful for quick culling.
7
Export
Choose a filename prefix and format, then hit EXPORT KEPT. You'll be prompted to pick a save folder — images go straight there.
⚠ No files are ever deleted from your disk — this tool only manages what you export.
⚠ TIFF files are not supported — convert to JPG or PNG first.
⚠ All processing is local. Nothing leaves your device.

How To Use dPaul TTS

☁ Google TTS

  • Get an API key — free, via the "How to get a free API key" link in that panel. Covers 1 million characters a month at no cost.
  • Paste and save your key — it's stored only in your browser, never sent anywhere else.
  • Pick a voice type — WaveNet and Neural2 sound the most natural. Studio is premium (paid beyond free tier). Standard is more robotic but free.
  • Choose a language and specific voice, then type your script and hit SPEAK.

B Browser TTS

  • Uses your computer's own built-in voices — no API key, no setup, works instantly offline.
  • Sounds noticeably more robotic than Google TTS or My Voice, but it's the fastest option with zero cost.
  • Adjust rate and pitch with the +/− controls, then hit SPEAK.

V My Voice

  • Start the local server on your Mac first (the Python server running in Terminal) — this page can't generate audio in your voice unless it's running.
  • Check the connection dot next to "Local Server Address" — green means connected, red means the server isn't reachable.
  • Record a voice sample with the RECORD NEW SAMPLE button — speak naturally for 10–30 seconds in a quiet room. You can record and keep several samples; more varied samples generally produce a richer, more natural clone.
  • Delete unwanted samples anytime with the ✕ next to each one in the list.
  • Type your script in the SCRIPT box, then hit SPEAK. A "Generating..." popup will appear — this can take 10–30 seconds since it's running on your Mac's processor, not a fast cloud server.

Speed & Expression Sliders

  • These only affect the next generation — moving a slider does not change audio that's already been generated or is currently playing.
  • Speed — how fast the voice talks. 1.00 is normal pace.
  • Expression — how much natural variation and inflection the voice has. Lower values sound flatter and more consistent; higher values sound more animated but slightly less predictable.
  • To hear a change, adjust the slider, then click SPEAK again.
Tip: keep the script reasonably short per generation (a paragraph or two) rather than pasting huge blocks of text — shorter passages generate faster and are easier to review before you commit to a longer piece.
How To Get A Free Google API Key
1
Create a Google account You need a Gmail account to use Google Cloud. If you already have one (Gmail, Google Drive, etc.), you can skip this step.
2
Go to the Google Cloud Console Visit console.cloud.google.com and sign in with that Google account. You may be asked to accept their terms and create a project — any name works, e.g. "dPaul TTS".
3
Enable the Text-to-Speech API In the search bar at the top, search for "Text-to-Speech API", open it, and click Enable. This is what allows your project to actually generate speech.
4
Create an API key Go to APIs & Services → Credentials in the left menu, click "Create Credentials", then choose "API Key". A long string of letters and numbers will appear — that's your key.
5
Copy the key into this app Paste it into the "Google Cloud API Key" field and click SAVE. It's stored only in your browser — never sent anywhere except directly to Google when you generate speech.
Free tier covers 1 million characters per month, which is more than enough for most personal use. Google will ask for a billing card on file, but you won't be charged unless you go over the free allowance — worth setting a budget alert in the Cloud Console if you want extra peace of mind.
